\caption{%
  \textbf{Surgical guidance} applies decomposed classifier-free guidance
  selectively within the semantic bifurcation window
  ($p \!\in\! [p_s, p_e]$), while using standard joint guidance outside.
  The strategy is model-agnostic: it applies identically to
  CFM (velocity $v$, forward integration) and
  DDIM (noise $\epsilon$, backward integration),
  differing only in the predicted quantity and integration direction.
  $\varnothing$ denotes the null token (label 9) used for classifier-free guidance.%
}
